下载 OpenVPN 配置文件 (.ovpn 文件)

你可以下载 .ovpn 文件连接此 OpenVPN server。
使用以下 .ovpn 文件之一。参考提示选择一个。

在某些国家，你必须指定的目标 VPN 服务器的 IP 地址，而不是 DDNS 主机名。

  • 如何安装建立 OpenVPN Client

 

包含 IP 地址的 .ovpn 文件

目标 IP 地址: 60.27.163.87

  • OpenVPN 配置文件: 60.27.163.87 (UDP 1690)

  • OpenVPN 配置文件: 60.27.163.87 (TCP 1429)

 

提示

一般来说，推荐使用包括 DDNS 主机名的 .ovpn 文件。DDNS 主机名是非常方便的，因为当相应的 IP 地址改变时， DDNS 主机名记录的 IP 地址也将被自动改变。不过，从某些国家或地区， DDNS 服务可能无法访问。在这样的环境中，域名 ".opengw.net" 不能被解析。您应该使用包含有 IP 地址的 .ovpn 文件。

如果同时有 TCP 和 UDP 连接，你可以使用其中之一。通常，端口号小于 2000 的主机，可以稳定地接受 VPN 连接。 TCP 与大多数防火墙是比较熟的，但是 UDP 可以实现更快的吞吐量。如果可以使用 UDP ，使用 UDP。

 

连接失败 ?

如果你不能用此 .ovpn 文件连接，请尝试其他 VPN server。
